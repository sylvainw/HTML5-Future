Clay.io API

Objectif : permettre de gérer des services pénibles en peu de lignes de code

  • Réseaux sociaux
  • Leaderboards et achievements
  • Statistiques
  • Micro-paiements

PeerJS API

  • Échange de données à travers un réseau P2P (WebRTC en Working Draft au W3C)
  • Connexions, sérialisation des données, ...
  • Compatibilité : Chrome 26, Firefox 22

PeerJS : local

Install the library 
> npm install peer

Create the server

Run the server 
> nodemon server.js

PeerJS : démo(s)

  • Clay.io : login via Facebook/Twitter
  • PeerJS : connexions temps-réel
  • Google text-to-speech
  • Google Maps : géolocalisation des résultats
